Sistema de gerenciamento de funcionários no Python

É possível ver na prática o processo de cadastro de funcionários utilizando o teclado como entrada de dados. com a opção '1' 'cadastrar funcionario' o sistema estebelece um id para aquele novo funcionario e pede ao usuario digitar o nome, setor e salario do funcionario, depois do 'enter' o sistema adiciona o novo funcionario a uma lista que pode ser consultada depois na opção '2' no menu principal. para remover um funcionario basta digitar o codigo que foi atribuido a este
